### hand-me web site

## Getting started

# Clone and setup the project
```
git clone https://gitlab.com/n19l/master_gpe.git && git checkout develop_front && cd hand-me-front/ && npm install
```

# Run the procject
```
npm run dev
```

## Compiles and minifies for production

npm run build
